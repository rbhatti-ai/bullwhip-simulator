# Bullwhip Effect Simulator

Supply chain learning simulator for university courses.

## Quick Start

**Backend** (Terminal 1):
```bash
cd backend
pip3 install fastapi uvicorn python-multipart
python3 beer_game_backend.py
```

**Frontend** (Terminal 2):
```bash
cd frontend
npm install
npm start
```

## Architecture

- React frontend on port 3000
- Python FastAPI backend on port 8000
- 4-tier supply chain simulation
- Demonstrates bullwhip effect

## Deployment

- Frontend: Vercel
- Backend: Railway
